# E2EE Clock

Simple watchface featuring an "end-to-end encrypted" (E2EE) style clock!

- [x] Bold Aesthetic: Four striking digits rendered through a random ciphertext effect. 🔐
- [x] Smooth Motion: Easily configurable with a beautiful encryption animation.

I created this watchface to raise awareness for end-to-end encryption—the only standard that truly protects our privacy.

Let’s make end-to-end encryption the default! 🔒

Made with ♥️ in Munich, Germany.

# Development

This project has two pebbleOS watchface subprojects:
### pebbleface-e2ee-clock-js
- implemented using the new alloy javascript SDK for PebbleOS
- feautures a beatiful encryption animation
- supports Pebble Time 2 and Pebble Round 2
### pebbleface-e2ee-clock-c
- implemented using the standard c SDK for PebbleOS
- supports older Pebbles, like Pebble Time, etc.
